# Fast Downloader
This is a fast YouTube downloader written in Electron. 

It currently supports:
- Windows 10/11
- Ubuntu/Debian
- CentOS/Red Hat
- Arch Linux

If your system isn't directly supported, you can try with the `AppImage`.
